Place character art and voice clips here so Card.ResolveAssets() can find them.

Folder layout (example):
  Resources/Characters/Priest/portrait.png   (Texture Type: Sprite)
  Resources/Characters/Priest/voice.wav

JSON paths omit the extension and the "Resources/" prefix:
  "portraitResourcePath": "Characters/Priest/portrait"
  "voiceResourcePath": "Characters/Priest/voice"
